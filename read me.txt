1. Food

POST /foods/ → Add a food item.

GET /foods/ → Retrieve all food items.

2. Daily Logs

POST /logs/ → Add a daily log (make sure food_id exists).

GET /logs/{log_date} → Get logs for a specific date.

3. Daily Totals

GET /totals/{log_date} → Check calorie/protein/carbs/fats totals.

4. User Goals

POST /goals/ → Set goals (ensure keys match schema: calories, protein, carbs, fats).

GET /goals/ → Retrieve goals.

5. User Profiles

POST /profiles/ → Create profile.

GET /profiles/ → List all profiles.

GET /profiles/{profile_id} → Get profile by ID.
